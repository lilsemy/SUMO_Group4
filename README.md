To run the program, you have to either run the Unix_Exe.jar or WIN_Exe.jar (depending on your machine)

How to run via terminal:
1. Go to directory of the project
2. java-jar <Unix_Exe.jar / WIN.Exe.jar>

*WIN_Exe.jar is currently not working.

The .zip contains the following files:
- executable .jar files for our project
- Project Documentation (containing the declaration of authorship)
- Program folder (src, lib, target)
